# CaiLama-Search Benchmark: Lexical vs Hybrid

## Metadata

- **Date:** 2026-05-23
- **Git-Ref:** `90997a5` (CaiLama-Search, nach Filter- und Multi-Index-Fixes)
- **Dataset:** goldsets (4 files, 9 cases, 8 fixtures)
- **Models:** all-MiniLM-L6-v2 (384-dim, sentence-transformers, cpu)

## Comparison

| Metric | Lexical | Hybrid |
|--------|---------|--------|
| Pass Rate | 100.00% | 100.00% |
| MRR | 0.9167 | 0.9167 |
| Recall@5 | 1.0000 | 1.0000 |
| Recall@10 | 1.0000 | 1.0000 |
| Zero-Hit Rate | 11.11% | 11.11% |
| Latency p50 | 0.0ms | 0.0ms |
| Latency p95 | 1.0ms | 0.0ms |
| Cases Passed | 9/9 | 9/9 |

## Analysis

- **Recall ist in beiden Modi identisch (Recall@5=1.0, Recall@10=1.0).**
  Die synthetischen Fixtures sind so klein (8 Docs), dass beide Modi
  alle relevanten Dokumente finden.
- **MRR ist in beiden Modi identisch bei 0.9167.**
  Der bewusst negative Zero-Hit-Case verhindert eine perfekte MRR.
- **Pass Rate ist in beiden Modi 100%.**
  Der fruehere `filter+hybrid`-500er ist behoben; der Hybrid-Pfad nutzt keine
  konfliktierende Meilisearch-`hybrid`-Option mehr, sondern Vector Search mit
  durchgereichtem Filter.
- **Der Multi-Index-Pfad ist normalisiert.**
  Gruppierte Search-/DWZ-Antworten liefern wieder die Pflichtfelder
  `query`, `items`, `results` und `processingTimeMs`.
- **Latenz ist in beiden Modi nahezu identisch** (sub-Millisekunde).
  Bei 8 synthetischen Docs ist der Embedding-Overhead nicht messbar.

## Conclusion

Lexical und Hybrid sind auf dem aktuellen kleinen synthetischen Goldset
korrektheitsparitaetisch. Die Gewinnschwelle von >=5% Recall@5 Verbesserung
wird nicht erreicht (beide 1.0000). Der Default `semantic.enabled=false`
bleibt deshalb empfohlen, bis ein groesseres Eval-Set einen produktiven Nutzen
zeigt.

## Notes

- Synthetic goldset fixtures; no production data used.
- No secrets, raw queries or private texts included.
- Goldset dataset (8 fixtures) ist zu klein für statistisch belastbare Aussagen.
- Für produktive Aussagekraft: >100 echte Dokumente und diverse Queries nötig.
- DWZ-Staging-Import und produktive Hybrid-Freigabe bleiben separate
  Folgeentscheidungen.
